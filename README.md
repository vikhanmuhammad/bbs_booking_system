Overview
BBS Booking System is a mobile application developed for BBS Pool & Cafe to streamline the billiard table reservation process. Built with Flutter for a smooth cross-platform experience and powered by Firebase for a robust backend, this app simplifies bookings for both customers and administrators.

Key Features
ðŸ‘¤ For Customers
Direct Booking: Easily browse available tables and book a time slot directly from the app.

Booking History: View past and upcoming reservations.

Real-time Availability: Check the availability of billiard tables in real-time.

ðŸ‘‘ For Admin
Reservation Management: View a centralized dashboard of all customer bookings to manage and confirm reservations.

Operational Control: Easily open or close the service based on operational hours, preventing bookings outside of business time.

Customer Data: Access booking details to provide better customer service.

Technology Stack
Frontend: Flutter

Backend: Firebase

Firestore: For real-time database management (bookings, user data, table status).

Firebase Authentication: For secure user login and registration.

BBS Booking System adalah aplikasi mobile yang dikembangkan untuk BBS Pool & Cafe guna mempermudah proses reservasi meja biliar.
